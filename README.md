# SmartQuiz AI : Gamified learning and quiz platform
<br>
Authors- Adithi Upadya, Sameeksha, Jiya Shetty
<br>
SmartQuiz AI - Gamified learning and quiz platform
<br>
SmartQuiz AI is an interactive gamified learning platform that improves student engagement through quizzes, streaks , leaderboards and adaptive difficulty.It also provides a teacher dashboard to track student performance and their activity.
<br>
<b>Important features of our solution is:</b>
<br>
<b>Student Side</b> <br>
Student login,
Personalized welcome dashboard,
  <br>
Sidebar with:
<ol>
<li> Profile</li>
<li>Leaderboard</li>
<li>Quizzes</li>
<li>Weekly quests</li>
<li>Activity</li>
<li>Time spent</li>
<li>Streak tracking</li>
  </ol><br>
Start quiz button,
Adaptive quiz difficulty (basic AI logic),
Instant feedback,
Streak system .
<br>

<b>Teacher Side</b> <br>
Teacher login, View list of students ,Click a student to see:
<ol>
  <li>Quiz scores</li>
<li>Streak</li>
<li>Time spent</li>
<li>Activity</li>
<li>Badges</li>
<li>Visual attendance streak matrix</li>
</ol>
